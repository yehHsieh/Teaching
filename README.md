
## 目的

- 提供家教課程介紹平台

## 功能

- 家教課程簡介介面
- 可觀看課程介紹

## 技術
- hover 的操控
- 設計 RWD 
- BS5 框架
- 引入 Swipe 套件，並客製化

#### 資料夾結構

- App # 原始碼
  - assets # 靜態資源放置處
    - images # 圖片放置處
    - js # JavaScript 放置處
    - style # 樣式放置處
  - index.html # 首頁 HTML
  - layout.ejs # Layout ejs
  - class.html # 課程 HTML
- gulpfile.js # Gulp 原始碼
  - envOptions.js # Gulp 路徑變數
  - index.js # Gulp 核心原始碼
